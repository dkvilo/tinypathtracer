# Tiny Raytracer

This is simple software Pathtracer written in C++ using SDL2.

Camera Controls:

- W/A/S/D
- SPACE - up
- CTRL - down
- Mouse (yaw and pitch)

- B - Visualize BVH nodes (Mesh and Sphere) (control division by + and -)
- T - Visualize Wireframe (Mesh)
